Краткая сводка по таблицам базы данных Northwind:
Categories - содержит информацию по ID, наименованиям категорий продуктов, описанию продуктов и изображения;
Contacts -  содержит полную информацию о контактах заказчиков, поставщиков, сотрудников, грузоотправителей с именами, контактными лицами, месторасположением;
CustomerCustomerDemo - содержит пустые столбцы с ID заказчиков и их типами (вероятно, должна заполняться на основе таблицы Customers);
CustomerDemographics - содержит пустые столбцы с ID типов заказчика и их описанием (вероятно, должна заполняться на основе таблицы Customers 
и таблицы CustomerCustomerDemo);
Customers - содержит подробную информацию о заказчиках, их ID, наименование, контактные данные, адреса (на основе этой таблицы можно заполнить
две предыдущие);
Employees - таблица содержит подробную информацию о сотрудниках, их ID, наименование, контактные данные, адреса;
EmployeeTerritories - устанавливает связь между ID сотрудника и ID территории (на основе таблиц Employees и Territories);
Order Details - содержит информацию о заказах, ID заказа, ID продукта, стоимость и количество продуктов в заказе, скидку (имеет связь с таблицей Products);
Orders - содержит подробную информацию о заказах, заказчиках, сотрудниках, датах заказа и доставки, адресах;
Products - содержит подробную информацию о продуктах, их наименование, стоимость, к какой категории относятся, цена, количество на складе и пр.;
Region - содержит информацию по ID региона и его описание;
Shippers - содержит информацию о грузоотправителях, их ID, наименование и телефон;
Suppliers - содержит подробную информацию о поставщиках, ID, наименование, контактных лицах и данных, месторасположении;
Territoties - содержит информацию о территории, ID, описание териитории, ID региона.
